# MAX 2.0 — Command Center

MAX is the AI assistant and guru. Greets visitors in their language and helps with whatever they need.

**Features:** Multi-language (EN, FR-QC, ES, PT-BR) · Adaptive help (Lawyer, Plumber, Therapist, Diet/Fitness, Life Coach, Advertiser) · Never pushy · Never forgets

---

## Quick Start

Open any HTML file in a browser. No build step, no npm required.

```bash
start index.html          # Windows
open index.html           # macOS
```

Or serve via static server:

```bash
npx serve .
```

## Pages

| File | Purpose |

| `index.html` | Marketing site — hero, features, pricing, Agent Swarm |
